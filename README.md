# Battleship Game - React Native Project

## Features

- **Authentication**:
  - Login
  - Register

- **Routing**:
  - Navigate between app screens

- **User Details Screen**:
  - View user details

- **Lobby**:
  - View all games
  - Join existing game
  - Create your own game
  - Game status indicator

- **Gameplay**:
  - Ship selection UI
  - Placement of ships on the board
  - Send configuration to API
